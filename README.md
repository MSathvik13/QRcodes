# QRcodes
Interesting simple python 3 programs to get acquainted qrcode module of pypi. 

Interesting simple python 3 programs to get acquainted qrcode module of pypi. 
There 3 programs first one deals with how to encode data into a QR (quick read) code. 
The second one is about how we can read a QR code and decode it. 
And the final one tells us how we can read QR code and decond it via webcam.
